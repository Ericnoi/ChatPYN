# 项目介绍

本程序是一个简单的人机对话程序，你可以随心所欲的发言，本程序中的智能体——PYN也会从已有的语料库中寻找句子，“热情”地给予回应！

本程序的原型是宜昌某初中英语老师（Jenny Pei），由于她的发言十分幽默有趣、富有启发意义，给一众学生留下了深刻印象，故开发此程序，以示敬意。

本程序为开源项目，遵循 MIT 协议。

# 使用指南

当程序开始时，会以 ">>>" 作为提示符提醒输入。用户只需输入自己想说的话，等待智能体回复即可。随后可以不断发起新一轮对话。
